📕Role
----------------------------------------------
1. 아침에 프로젝트 시작하기 전에 전부 일괄적으로 Pull 받고 시작하기
2. Push하기 전에 단톡방에 Push한다고 말하고 하기
3. Commit 메시지 본인이름쓰고 수정한 내용 상세히 적기
  ex) 유재열 : 비밀번호 찾기 인증 구현
4. 코드 변동사항 많을 때 따로 복사본에 테스트하기
5. 팀원들과의 소통을 원활하게 하여 겹치는부분 없이 하기
