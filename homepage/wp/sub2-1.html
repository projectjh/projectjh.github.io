UEBA (User and Entity Behavior Analysis)

What is WatchPoint UEBA?

WatchPoint UEBA is the monitoring system that could monitor and predict the risks of internal information leak in real time by forming the profile of each user and analyzing the behavioral patterns through the application of information leak detection monitoring framework.

Major functions

01Interlocked with legacy system(HR, absenteeism & tardiness, SMS, and etc.)

 On top of security devices, it also provides the detailed and accurate analysis results by collecting/analyzing the log of HR and absenteeism & tardiness system. In case when an event occurs, it is immediately transmitted to the manager through email and SMS.

02Log interlocking & log normalization such as various work systems & security solutions

 The unified user-centered behavior basis is analyzed by relocating the collected log information into user-centeredness.

03Function to automatically detect abnormal signs besides general scenario(profiling)

 It is possible to provide the abnormal behavior profiling in the unit of department/individual. It supports the multidimensional analysis function applying the hybrid-profiling technique that provides the statistical accumulation—based profiling technique and the statistics modeling-based profiling technique at the same time.

04Provision of correlation/linkage analysis function of various types of log

 It provides the inter-system correlation/linkage analysis function of various types of log.

05Vindication process

 The vindication process has been built in to provide the detailed functions necessary for vindication, so the object of vindication could vindicate oneself on email with no necessity of separate log-in process.

06If necessary, it is interlocked with the original file such as DRM file/Print/HTML.

 For the verification of abnormal behaviors, the efficient control work is supported through the linkage with the original file such as DRM file / printer image & text / DLP file.
